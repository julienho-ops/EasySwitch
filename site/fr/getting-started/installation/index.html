    Install EasySwitch¶

    Requirements¶

    • Python >=3.9
    • Package manager (pip or uv recommanded)
    • API credentials for any supported payment provider (PayGate, FedaPay, etc.)

    Install¶

    pip install easyswitch

    # First install UV
pip install uv

# Then install EasySwitch
uv pip install easyswitch

# Or 
uv add easyswitch

    git clone https://github.com/your-repo/easyswitch.git
cd easyswitch
pip install -e .[dev]  # Development mode

    Check installation¶

    import easyswitch
print(easyswitch.__version__)
# Example output: '1.0.0'
